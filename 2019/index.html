为什么弃用WordPress并搬离阿里云

总的来说，原因有三：

  1. 主机频繁死机
  2. WordPress 越来越臃肿，不想花更多钱升级服务器
  3. 要备案

半年前开始，部署在阿里云虚拟主机的 WordPress 博客和 awildpetcat.com的留言本，可用性频繁降到0，7月27日这天就有三次挂，短信通知邮件通知，手机电脑就是一通叮铃咣啷。

阿里云独享虚拟主机经济版：

空间：5G CPU：单核 内存：1G 带宽：1Mbps MySQL：500MB

2016年5月刚把网站从万网搬到阿里云之初，用起来感觉很好，可好景不长。之前操作换过一次主机机房，仍然出现这问题。

这锅也不能阿里云一个人背，Wordpress 功能越来越臃肿也是原因。最近 WordPress 更新的新版编辑器给我的体验还挺好，我另外安装插件Akismet Anti-Spam, Jetpack, WP Super Cache，也足够满足博客的需要。还有种类繁多的主题，可以实时预览和安装。那么臃肿也是这么多好玩、强大功能的代价吧。但是仔细想我对博客的需求，其实并不需要这么多。

最后一个原因是上个月，我备案所属辖区片儿警给我打电话，让又重新注册备案一个什么编号。这些鬼 big brother 真的很烦人，很烦人！

现在 Github Pages, Netlify 等服务免费又方便，纯静态也能满足需要，与其买每年298元的阿里云，还不如搬家咯，准备在 Hexo 和 Gatsby 之中选一个迁移走。

WordPress 迁移

考察了 gatsby-source-wordpress 插件后，觉得太麻烦，准备用 WordPress 插件 WP2Static 将全站导出为静态文件，然后融合到新站中，同时域名也可以保持不变。

小米电动滑板车开箱与感受

在腾讯视频观看，在优酷观看

单位年会抽奖中了小米电动滑板车，开箱使用过3、4回后说说体验，除了充电的几个问题（充电接口位置、充电指示），给我的总体感觉是比较满意的。

官方指标参数

  • 最高车速25km/h
  • 推荐最大载重75kg
  • 平路路况下最大续航30km
  • 整车重12.5kg
  • 充满电约5小时

在实际测试中车速极速模式最高可达26.3km/h，节能模式20.2km/h。

相比更小巧的电动滑板，小米这款尺寸算很大了，更大的尺寸意味着，有更多空间可以实现更好的功能设计，其中的蓝牙连接扩展和碟刹、能量回收双制动是最大的亮点。

蓝牙连接扩展可以在米家App中，查看滑板车车速、电池、续航里程信息，设置定速巡航、尾灯、能量回收强度、电机密码锁，还可以给电动车进行固件升级，可玩性大大提高，不过也就图个新鲜罢了，玩过之后估计也没谁天天把玩这几个设置。

带能量回收的双重刹车，是个很实用的设计，在App中设置能量回收强度为强，刹车时轻捏刹车使用电机制动，大多数时候碟刹都用不到，减少碟刹的磨损同时增大续航里程

虽然用上了充气的轮胎，和较大的轮胎尺寸，但过坎和不平整的地面，还是要注意减速并抓紧龙头，20几的时速还是有点小怕的。

整车12.5kg重量了的，女生搬稍有点吃力。但更大的尺寸意味着更大的马力和使用舒适度，在小区附近买东西、工业园区代步、公园游览都是不错的使用场景。

缺点

好，最后是我觉得两个不尽人意的地方：

  • 充电接口位置太低，要趴下来才看得见，摸着操作不容易找到
  • 充电不开电动车电源时，完全看不出来任何指示，要打开电源，才能看到电量指示灯的闪动和亮起尾灯。

使用建议

  • 在城市路况复杂的情况，多使用“节能模式”，以限制最大车速；
  • 多使用“能量回收制动”，增大巡航里程；
  • 少爬陡坡，减少电机、电池高负荷工作，从而延长使用寿命；
  • 下雨天不要使用，后面的挡泥板不好使，会甩一后背的泥巴。
