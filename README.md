# 旧 Wordpress blog 存档
[https://archived-blog.berlinchan.com](https://archived-blog.berlinchan.com)  
从旧 Wordpress blog 迁移的站点，由插件 WP2Static 生成并精简处理  

## Tools
tools 目录下存放批处理工具
- removeUnusedSVG.js, 移除 html 中多余的 svg tag
- encrypt.js, 内容加密
